Barley

The UK Barley Genome Sequencing consortium is a joint effort of the James Hutton Institute (JHI), The Genome Analysis Centre (TGAC), and Ensembl Genomes, funded by grants from the BBSRC and the Scottish Government. The consortium aims to create a comprehensive set of genomic resources for Barley. These include:

  • A complete, ordered gene-space assembly.
  • A deep transcriptome reference data set.
  • A genome-wide set of informative sequence variants.

Additionally, the consortium contributes to the International Barley Genome Sequencing Consortium's efforts to develope a complete barley genome sequence.

The genomic resources currently available in Ensembl Plants are based on the latest gene-space barley assembly from the International Barley Genome Sequencing Consortium [1].

Oryza sativa Japonica vs Solanum tuberosum LastZ alignments

Oryza sativa Japonica (Oryza sativa, IRGSP-1.0) and Solanum tuberosum (Solanum tuberosum, 3.0) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Oryza sativa Japonica was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

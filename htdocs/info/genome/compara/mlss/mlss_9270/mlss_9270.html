Oryza punctata vs Oryza glumaepatula LastZ alignments

Oryza punctata (Oryza punctata, AVCL00000000) and Oryza glumaepatula (Oryza glumaepatula, ALNU02000000) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Oryza punctata was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

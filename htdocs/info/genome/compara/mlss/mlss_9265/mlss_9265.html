Oryza barthii vs Oryza sativa Indica LastZ alignments

Oryza barthii (Oryza barthii, ABRL00000000) and Oryza sativa Indica (Oryza indica, ASM465v1) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Oryza barthii was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

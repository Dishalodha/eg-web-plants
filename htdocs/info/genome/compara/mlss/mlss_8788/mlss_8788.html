Vitis vinifera vs Solanum lycopersicum LastZ alignments

Vitis vinifera (Vitis vinifera, IGGP_12x) and Solanum lycopersicum (Solanum lycopersicum, SL2.40) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Vitis vinifera was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

Oryza punctata vs Oryza glaberrima LastZ alignments

Oryza punctata (Oryza punctata, AVCL00000000) and Oryza glaberrima (Oryza glaberrima, AGI1.1) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Oryza punctata was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

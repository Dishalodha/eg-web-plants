Hordeum vulgare vs Triticum urartu LastZ alignments

Hordeum vulgare (Hordeum vulgare, 030312v2) and Triticum urartu (Triticum urartu, GCA_000347455.1) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Hordeum vulgare was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

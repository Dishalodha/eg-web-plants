Arabidopsis thaliana vs Oryza brachyantha LastZ alignments

Arabidopsis thaliana (Arabidopsis thaliana, TAIR10) and Oryza brachyantha (Oryza brachyantha, Oryza_brachyantha.v1.4b) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Arabidopsis thaliana was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

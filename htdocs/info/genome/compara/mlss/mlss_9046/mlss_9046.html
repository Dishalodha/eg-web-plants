Brachypodium distachyon vs Hordeum vulgare LastZ alignments

Brachypodium distachyon (Brachypodium distachyon, v1.0) and Hordeum vulgare (Hordeum vulgare, 030312v2) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Brachypodium distachyon was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.

Oryza barthii vs Oryza meridionalis LastZ alignments

Oryza barthii (Oryza barthii, ABRL00000000) and Oryza meridionalis (Oryza meridionalis, ALNW00000000) were aligned using the LastZ alignment algorithm (LastZ) in Ensembl release 76. Oryza barthii was used as the reference species. After running LastZ, the raw LastZ alignment blocks are chained according to their location in both genomes. During the final netting process, the best sub-chain is chosen in each region on the reference species.
